An Android app for UCSC students to easily retrieve real-time course data as well as plan schedules for each academic term. This app is currently in haitus.

ResumeRater

A web application built to help specifically students review and manage their resumes. It will include tutorial videos for students just beginning their resumes and more specific tips for different fields. It will also include templates to start off or improve their current resumes. It's main feature is a star rating system which gives priority to more helpful reviews and to more experienced students or workers. This application is still in progress.
